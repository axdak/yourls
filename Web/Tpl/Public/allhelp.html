什么是短网址？

短网址服务可以帮助你把一个长网址缩短，方便你在社交网络和微博上分享链接。

例如，你可以把这样一个长网址

http://image.baidu.com/i?ct=503316480&z=0&tn=baiduimagedetail&word=%B6%CC%CD%F8%D6%B7&in=25321&cl=2&lm=-1&pn=3&rn=1&di=4944614685&ln=1&fr=&ic=&s=0&se=&sme=0

缩短成这样一个短网址

{:parseDomain()}sample

支持任意格式后缀访问，例如：

{:parseDomain()}sample.aspx

相对于长网址，短网址可以更方便地在电子邮件、社交网络、微博和手机上传播，避免折行或超出字符限制。

怎样生成一个短网址？

我们提供多种方式使用我们的短网址服务：

1.访问短网址首页http:///，输入你要缩短的原网址，生成对应的短网址。你还可以登录会员中心自定义你想要的短网址字符

2.使用短网址API开发的各种应用，调用短网址服务。例如书签、桌面程序、小插件等。

怎样分享一个短网址？

我们提供多种方式分享短网址方式：

1.生成短网址后会弹出生成结果窗口，你可以直接下载二维码发给你的朋友，比如拿微信扫一下，接受短网址信息。

2.生成短网址后会弹出生成结果窗口，你可以直接复制或者点击下方的分享按钮，分享给你的朋友。

怎样还原一个短网址？

当你看到一个已生成的短网址，想知道原来的网址是什么，怎样还原呢？

在短网址的“还原网址”页面输入你要还原的短网址，点击“还原”按钮，就可以看到对应的原始网址

怎样调用短网址API？

生成短网址时：

请求：向http:///index.php?m=Index&a=create发送post请求，发送数据包括url=长网址

返回：json格式的数据

1.status!=0 出错，查看err_msg获得错误信息（UTF-8编码）

2.成功，返回生成的短网址 tinyurl字段

显示原网址：

请求：向http:///index.php?m=Index&a=query发送post请求，发送数据包括tinyurl=查询的短地址

返回：json格式的数据

1.status!=0 出错，查看err_msg获得错误信息（UTF-8编码）

2.成功，返回原网址 longurl字段

 

示例程序：

 

生成短网址

 

<?php

$ch=curl_init();

curl_setopt($ch,CURLOPT_URL,"http:///index.php?m=Index&a=create");

curl_setopt($ch,CURLOPT_POST,true);

curl_setopt($ch,CURLOPT_RETURNTRANSFER,true);

$data=array('url'=>'http://www.pengyong.info');

curl_setopt($ch,CURLOPT_POSTFIELDS,$data);

$strRes=curl_exec($ch);

curl_close($ch);

$arrResponse=json_decode($strRes,true);

if($arrResponse['status']==0)

{

/**错误处理*/

echo iconv('UTF-8','GBK',$arrResponse['err_msg'])."n";

}

/** tinyurl */

echo$arrResponse['tinyurl']."n";

?>
